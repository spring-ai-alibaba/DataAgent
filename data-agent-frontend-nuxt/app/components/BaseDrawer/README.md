# 组件: BaseDrawer

## 模块描述
基础侧边栏布局组件，提供左侧抽屉和右侧内容区的响应式布局

## Props
| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| drawerWidth | `union` | 260 | 抽屉宽度 (数字或带单位的字符串) |
| modelValue | `boolean` | true | 抽屉是否展开 (支持 v-model) |

## Slots
| 名称 | 描述 |
| --- | --- |
| drawer | - |
| header | - |
| default | - |

## Events
| 名称 | 描述 |
| --- | --- |
| update:modelValue | 更新展开状态 |
