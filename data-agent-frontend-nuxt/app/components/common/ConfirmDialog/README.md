# 组件: ConfirmDialog

## 模块描述
通用确认对话框组件，用于二次确认操作

## Props
| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| modelValue | `boolean` | - | 是否显示对话框 |
| title | `string` | - | 对话框标题 |
| message | `string` | - | 提示消息内容 |
| prependIcon | `string` | 'mdi-help-circle' | 标题前的图标 |
| confirmText | `string` | '确认' | 确认按钮的文字 |

## Events
| 名称 | 描述 |
| --- | --- |
| update:modelValue | 更新显示状态 |
| confirm | 点击确认按钮事件 |
