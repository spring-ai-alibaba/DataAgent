# 简单数据查询

## 适用场景
- 用户提问包含"是多少"、"有多少"、"总计"、"数量"等关键词
- 查询结果只有少量数据（1-5 行）
- 用户不需要趋势分析或可视化图表

## 输出格式
直接用一句话回答，例如：
- "本月销售额为 ¥1,234,567"
- "当前注册用户共 8,932 人"
- "最高订单金额为 ¥9,800"

## 注意事项
- 保持简洁，不要附加额外分析
- 数字需要格式化（千分位、保留两位小数）
- 如果结果为空，说明"未查询到数据"
